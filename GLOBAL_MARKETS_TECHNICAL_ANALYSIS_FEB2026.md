### 1.2 四度空间理论核心概念

```
┌──────────────────────────────────────────────────────────────┐
│                    Market Profile 四度空间结构                  │
├──────────────────────────────────────────────────────────────┤
│                                                               │
│  【POC - Point of Control】                                   │
│    → TPO字母最多的价格水平                                    │
│    → 最大成交量区域，市场公认的"公平价格"                      │
│    → 价格偏离POC越远，回归倾向越强                             │
│                                                               │
│  【Value Area - 价值区域】                                     │
│    → 包含70%交易活动的价格区间（VAL-VAH）                      │
│    → 价格在价值区域内部→ 正常波动                             │
│    → 价格超出价值区域→ 回归概率高                             │
│                                                               │
│  【HVN - High Volume Node】                                   │
│    → 高成交量节点，强支撑/阻力                                 │
│    → 价格在此区域停留时间长，接受度高                          │
│                                                               │
│  【LVN - Low Volume Node】                                    │
│    → 低成交量节点，快速通过区域                                │
│    → 缺乏实质性支撑，价格快速移动                              │
│                                                               │
│  【TPO - Time Price Opportunity】                             │
│    → 时间-价格机会，每个字母代表一段时间内的价格活动           │
│    → TPO分布越广，市场接受度越高                                │
│                                                               │
└──────────────────────────────────────────────────────────────┘
```
