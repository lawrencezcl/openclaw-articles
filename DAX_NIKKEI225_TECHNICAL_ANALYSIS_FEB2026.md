## 第一部分：威科夫交易法分析框架

### 1.1 威科夫市场周期理论

威科夫方法将市场价格周期划分为四个关键阶段：

```
┌─────────────────────────────────────────────────────────────┐
│                    威科夫市场周期四阶段                         │
├─────────────────────────────────────────────────────────────┤
│  1. 吸累期 (Accumulation)     →  智能资金建仓，公众恐慌抛售    │
│  2. 上涨期 (Markup)           →  价格上升，公众开始关注        │
│  3. 分配期 (Distribution)     →  智能资金出货，公众FOMO买入     │
│  4. 下跌期 (Markdown)         →  价格下跌，周期循环重启        │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 威科夫三大定律

1. **供求定律**: 价格因供不应求而上涨，因供过于求而下跌
2. **因果定律**: 累积区间的幅度（因）决定后续上涨的幅度（果）
3. **努力与结果定律**: 上涨的成交量（努力）必须与价格涨幅（结果）相匹配
